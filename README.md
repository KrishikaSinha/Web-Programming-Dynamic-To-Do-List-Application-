# Dynamic To-Do List Application

## 📌 Project Overview

The **Dynamic To-Do List Application** is a full-stack project that allows users to manage their daily tasks efficiently. Users can add, view, update, and delete tasks in real time.

This project demonstrates the integration of **Frontend and Backend technologies** with a **database** to create a complete task management system.

---

## 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Dynamic task updates
* Data stored using SQLite database

---

## 🛠️ Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQLite

## ⚙️ How to Run the Project

### Step 1: Install Dependencies

Navigate to the backend folder and install required packages.

```
cd Backend
npm install
```

### Step 2: Start the Server

```
node server.js
```

### Step 3: Open Frontend

Open the **index.html** file in your browser to use the To-Do List application.
